## Run Locally

Clone the project

```bash
  git clone https://github.com/waqarkalim/ShopifyFrontEndDeveloperChallenge2022
```

Go to the project directory

```bash
  cd ShopifyFrontEndDeveloperChallenge2022
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Deployment

To deploy this project run

```bash
  npm run deploy
```
